Executive Summary

Executive Summary

Shadow Run is a privacy-native multiplayer game world built exclusively for the Midnight Network. It transforms complex cryptography and network infrastructure into an immersive, gamified consumer experience.

Instead of traditional decentralized exchange (DEX) interfaces or zero-knowledge (ZK) utility wrappers, Shadow Run turns Midnight's core primitives into lived user behavior. When developers look at Zswap, they see shielded UTXOs and atomic swaps. When players look at Shadow Run, they see Ghost Trails, River Crossings, and Stone Drops.

The Problem with Current "Privacy Finance"

In the contemporary Web3 landscape, every bridge, DEX, and DeFi protocol operates within a "glass house." All actions are fully public, transaction histories are traceable, and trading strategies are leaked instantaneously. Existing "gamified DeFi" (e.g., DeFi Land, Axie Infinity) gamifies the rewards, not the execution layer itself. The underlying mechanics remain completely transparent.

The Shadow Run Solution

Shadow Run uses Midnight's dual-state ledger, Compact smart contracts, and Zswap protocol to build a world where the transaction itself is the game.

There is no public visibility into:

  • The assets you hold
  • Your peer-to-peer trading routes
  • The endpoints of your transfers

There is only cryptographic proof of your proficiency—your ZK-verified Trail Score, recorded on a public leaderboard, stripped of any correlated financial telemetry.

We believe that privacy should not require a whitepaper to be understood; it should be felt within the first two minutes of gameplay.
